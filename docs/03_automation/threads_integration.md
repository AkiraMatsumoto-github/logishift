# Threads API 連携完全ガイド

このドキュメントでは、アカウント作成からトークン発行まで、Threads API連携に必要な全手順を解説します。

## 0. 前提知識とアカウント構造

Threads APIを使用するには、以下の3つのMetaエンティティの関係を理解する必要があります。

1.  **Facebookアカウント**: Meta Developerになり、アプリを作成・管理するために必要です。
2.  **Instagramアカウント**: Threadsプロフィールを作成するために必要です。
3.  **Threadsプロフィール**: 実際に投稿を行うエンティティです。

> [!NOTE]
> **重要:** 個人のプライベートアカウントとは別に「Bot」や「ブランド」のアカウントを作りたい場合でも、**それ専用のInstagramアカウントを新規作成する必要があります**。

### 必要な役割 (ロール)

| エンティティ | 役割 | API利用の要件 |
| :--- | :--- | :--- |
| **Facebookアカウント** | 開発者 (Developer) | アプリIDとシークレットキーを管理します。 |
| **Instagramアカウント** | Threadsユーザー | APIをスムーズに利用するには**プロアカウント（ビジネスまたはクリエイター）**である必要があります。 |
| **Metaアプリ** | プラットフォーム | 「テスター」という役割を通じて、開発者とユーザー（Threads）を接続します。 |

---

## 1. アカウント作成ステップバイステップ

既存の個人アカウントとは完全に分離された「ブランド/Bot用アカウント」を作成する手順です。

### 手順1: 新しいInstagramアカウントの作成
1.  Instagram（スマホアプリ推奨）またはWebにアクセスします。
2.  **新しいアカウントを作成**します（メールアドレスまたは電話番号で登録）。
    *   *ヒント: 個人用と分けるため、専用のエイリアス（例: `bot@yourdomain.com`）の使用をお勧めします。*
3.  希望するBot名のユーザーネームを設定します。
4.  **重要:** **プロアカウントに切り替えます**。
    *   `設定` -> `アカウントの種類とツール` -> `プロアカウントに切り替える` へ移動します。
    *   「ビジネス」または「クリエイター」を選択します（ブランド運用の場合は「ビジネス」推奨）。
    *   カテゴリを選択します（例: 「メディア/ニュース」「ブログ」など）。

### 手順2: Threadsプロフィールの有効化
1.  スマホに **Threadsアプリ** をダウンロードします。
2.  先ほど作成した **新しいInstagramアカウント** でログインします。
3.  プロフィール設定（自己紹介、リンク、アイコン）を完了させます。
    *   *注: このプロフィールが、APIが操作する法的な「ユーザー」となります。*

### 手順3: Meta Developerアカウント (未作成の場合)
*この手順は既存の個人用Facebookアカウントで行えます。「アプリ」はこのアカウントで管理しますが、「投稿者」は手順2のアカウントになります。*

1.  [Meta for Developers](https://developers.facebook.com/) にアクセスします。
2.  **メインのFacebookアカウント**でログインします。
3.  必要に応じてアカウント認証（携帯電話番号認証など）を行います。

---

## 2. アプリの作成と設定

### 手順4: アプリの作成
1.  Meta for Developersで **「マイアプリ (My Apps)」** -> **「アプリを作成 (Create App)」** をクリックします。
2.  **「ユースケース (Use Case)」** を選択: **Running a specific app?** -> **Threads** を選択します。
    *   *もし「Threads」が選択肢にない場合は、「その他 (Other)」 > 「ビジネス (Business)」を選択し、後で製品として「Threads」を追加してください。*
3.  アプリ名を入力します（例: "LogiShift Auto Poster"）。
4.  アプリを作成します。

### 手順5: Threads製品の設定
1.  アプリダッシュボードのサイドバーにある製品一覧から **「Threads」** を探して設定します（または「製品を追加」から追加）。
2.  **「開発モード (Development Mode)」** のままで使用するため、すぐに「アプリのレビュー」に出す必要はありません。

---

## 3. ユーザーとアプリの接続 (テスター招待フロー)

アプリが **「開発モード」** にあるため、明示的に招待したユーザー以外はこのアプリを使用できません。作成した新しいInstagram/Threadsアカウントを **「テスター (Tester)」** として招待する必要があります。

### 手順6: テスターの追加
1.  アプリダッシュボードで: **「アプリの役割 (App Roles)」** -> **「役割 (Roles)」** へ移動します。
2.  **「Threadsテスター (Threads Testers)」** という項目までスクロールします（重要: 単なる「テスター」ではなく、必ず「Threadsテスター」の項目を使用してください）。
3.  **「Threadsテスターを追加 (Add Threads Testers)」** をクリックします。
4.  新しいInstagramアカウント（手順1で作成）の **ユーザーネーム** を入力します。
5.  ユーザーを選択し、招待を送信します。

### 手順7: 招待の承認 (最重要！)
この手順を飛ばすとAPIは動作しません。

1.  **PCまたはスマホのブラウザで**、www.instagram.com に **新しいBotアカウント** としてログインします。
2.  `設定 (Settings)` (歯車アイコン) -> `アプリとWebサイト (Apps and Websites)` -> `テスターの招待 (Tester Invites)` へ移動します。
    *   *直接リンク:* [https://www.instagram.com/accounts/manage_access/tester_invites/](https://www.instagram.com/accounts/manage_access/tester_invites/)
3.  作成したMetaアプリからの招待が表示されています。 **「承認 (Accept)」** をクリックします。

---

## 4. トークンの発行 (自動化)

アプリとユーザーの関係が確立されたので（App <-> User）、トークンを発行できます。

1.  `.env` ファイルに、アプリダッシュボード（`アプリの設定` -> `ベーシック`）から取得した `THREADS_APP_ID` と `THREADS_APP_SECRET` が設定されていることを確認します。
2.  ヘルパースクリプトを実行します:
    ```bash
    python3 automation/auth_threads.py
    ```
3.  画面の指示に従います:
    *   表示されたURLにアクセスします。
    *   **Botアカウント**（手順2のアカウント）としてログインした状態でアプリを認証（許可）します。
    *   リダイレクトされたURL（またはコード）をスクリプトに貼り付けます。
4.  スクリプトが長期間有効な `THREADS_ACCESS_TOKEN` と `THREADS_USER_ID` を表示します。これらを `.env` に保存してください。

## 5. "Action Blocked" (アクションがブロックされました) エラーの対処

`Error 2207051: Action Blocked` や「コミュニティガイドライン」関連のエラーが出た場合:

### 原因
*   **新規アカウントフィルター:** APIを使用する作りたてのアカウントは、MetaのシステムからスパムBotと誤認されやすいです。
*   **アカウントの実績不足:** スマホでの「人間らしい」操作履歴が不足しています。

### 解決策
1.  **手動操作で「慣らす」:**
    *   スマホのThreadsアプリを開きます。
    *   手動で「Hello World」などを投稿してみます。
    *   いくつかの投稿に「いいね」をし、公式アカウント（例: @mosseri）などをフォローします。
    *   自己紹介文やアイコンが未設定なら設定します。
2.  **24〜48時間待つ:** 新規アカウントには厳しいレート制限があります。
3.  **設定の再確認:** Instagramアカウントが「ビジネス」または「クリエイター」になっているか確認します。
4.  **権限の確認:** Instagramの設定画面で「テスター招待」を正しく承認できているか再確認します。
